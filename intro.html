什么是中文编程？

我们不是把某种编程语言汉化，而是基于中文本身的特征，探索一种新的编程范式。我等道行有限，不敢奢望开天辟地，只求抛砖引玉。

基本语法

系统内部是一个基于模式匹配和转换的自动机器，其基本运行模式， 系统把每行字按照空格分成若干个词， 对句子的最右侧的词进行模式匹配，如果匹配成功，则将匹配成功的部分进行转换， 然后继续对转换后的句子最右侧的词进行匹配和转换，直到无法匹配为止。 甲 乙 丙 丁 戊 己 庚 辛 壬 癸 是特殊的关键词，用来表示变量

系统有一些内建模式，目前支持四则运算
甲 加 乙 => 结果
甲 减 乙 => 结果
甲 乘 乙 => 结果
甲 除 乙 => 结果

如果一个行里有 => 则这一行被认为是用户自定义模式转换规则，例如: 计算 甲 和 乙 的和 => 甲 加 乙
阶乘 1 => 1
阶乘 甲 => 甲 乘 阶乘 甲 减 一

如果一个行里没有 => 则系统会对这一行进行推演，并输出推演过程和结果。

把 甲 和 乙 相加 => 甲 加 乙 把 1 和 2 相加

示例程序 - 阶乘

阶乘 1 => 1 阶乘 甲 => 甲 乘 阶乘 甲 减 1 阶乘 4
